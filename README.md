## вңҰ What is Waza?

Waza is a personal journaling web app built as a calm, intentional digital space. It's designed for people who want more than a blank page вҖ” a guided experience that helps you show up for yourself every day.

Whether you're processing a hard day, setting a weekly intention, or simply checking in вҖ” Waza meets you where you are.

**Core pillars:**
- рҹ“қ **Journaling** вҖ” free-write or follow structured prompts
- рҹҺҜ **Goal Setting** вҖ” set daily intentions and to-do lists
- рҹ”Ғ **Daily Recaps** вҖ” reflect on what you accomplished and felt
- рҹҶҳ **SOS Contacts** вҖ” mental health resources always one tap away

## вҡЎ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18 or higher
- A [Firebase](https://firebase.google.com) project with **Email/Password** authentication enabled

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/waza-journal.git
cd waza-journal
```

### 2. Install dependencies

```bash
npm install
```

### 3. Configure Firebase

Create `src/firebaseConfig.js` and paste your Firebase project config:

```js
import { initializeApp } from "firebase/app";
import { getAuth } from "firebase/auth";

const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT.firebaseapp.com",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_PROJECT.appspot.com",
  messagingSenderId: "YOUR_SENDER_ID",
  appId: "YOUR_APP_ID"
};

const app = initializeApp(firebaseConfig);
export const auth = getAuth(app);
export default app;
```

> вҡ пёҸ Never commit real API keys to a public repository. Use `.env` variables in production.

### 4. Start the dev server

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## рҹҺЁ Design System

Waza uses a consistent earthy palette across all pages.

```js
// src/styles/colors.js
export const COLORS = {
  darkGreen:  "#606c38",   // Surfaces, panels
  forest:     "#283618",   // Page background
  cream:      "#fefae0",   // Primary text
  gold:       "#dda15e",   // Accents, CTAs, focus states
  clay:       "#bc6c25",   // Hover states, warmth
};
```

**Typography**
- Display / headings: **Quincy CF** вҖ” serious but with a whimsical warmth
- Body / UI: **Circular** вҖ” clean, readable, modern
